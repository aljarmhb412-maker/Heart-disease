# Heart Disease Classification

## Project Overview
This project uses Machine Learning techniques to classify whether a patient is at risk of heart disease based on medical attributes.

## Files
- Heart_Disease_Classification.ipynb : Data analysis, preprocessing, training, and evaluation.
- heart_data.xlsx : Dataset used in the project.

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results
The model was trained and evaluated to predict heart disease risk.
